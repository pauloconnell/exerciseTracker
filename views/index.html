Exercise Tracker


Simple app to create a user in the database with a unique userID and store exercises. Back End API set up to be querried with parameters as described at bottom of page.

Create a New User

Enter User Name and you will recieve your unique userID

Add exercises

Use your unique userID to log your exercises

ADVANCED USERS: exercise log: GET /api/exercise/log?{userId}[&from][&to][&limit]

{ } = required, [ ] = optional

from, to = dates (yyyy-mm-dd); limit = number

To get an Array of users get: /api/exercise/users/ OR CLICK HERE
